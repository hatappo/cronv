visualize crontab settings.

`rake demo`

```sh
$ rake demo

---------- sample crontab settings.
5-9,6,*/12 */2 * * * echo 123 > ~/foo.txt 2>&1
*/5        */4 * * * echo 456 > ~/bar.txt 2>&1
5,10,15      * * * * echo 789 > ~/baz.txt 2>&1
----------
```
